# 数学

考研数学总分类，包含高等数学、线性代数、概率论与数理统计三个子科目。

## 子科目

- [[高等数学]] — 极限、导数、积分、微分方程等
- [[线性代数]] — 矩阵、向量、线性方程组、特征值等
- [[概率论与数理统计]] — 随机事件、概率分布、数理统计等

## 考试结构

| 项目 | 内容 |
| --- | --- |
| 满分 | 150分 |
| 考试时长 | 180分钟 |
| 试卷类型 | 数学一、数学二、数学三 |

### 题型分布

| 题型 | 数量 | 分值 |
| --- | --- | --- |

| 填空题 | 6题 | 30分（每题5分） |
| 解答题 | 6题 | 70分（分值不等） |
