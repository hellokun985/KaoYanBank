# 数列的极限

## 定义
设 {xₙ} 为一数列，如果存在常数 a，对于任意给定的正数 ε（不论它多么小），总存在正整数 N，使得当 n > N 时，|xₙ - a| < ε 都成立，则称常数 a 是数列 {xₙ} 的极限。

记作：lim(n→∞) xₙ = a

## 收敛数列的性质

### 唯一性
收敛数列的极限必唯一。

### 有界性
收敛数列必有界。

### 保号性
若 lim(n→∞) xₙ = a > 0，则存在 N，当 n > N 时，xₙ > 0。

### 保序性
若 lim(n→∞) xₙ = a，lim(n→∞) yₙ = b，且 a > b，则存在 N，当 n > N 时，xₙ > yₙ。

## 子列
- 子列的极限若是原数列极限的子列
- 收敛数列的任一子列也收敛
